# Simple Chatbot using Python

A basic rule-based chatbot created as part of the CodeAlpha Internship task. It understands user inputs and responds with pre-defined replies using simple if-else logic.

## Features
- Greets the user warmly.
- Answers basic questions about well-being, time, and identity.
- Handles exit commands smoothly.

## How to Run
1. Make sure Python is installed.
2. Run the script via terminal:
   python chatbot.py
